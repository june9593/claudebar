# ClawBar — Product Requirements Document

> **角色**: Product Manager
> **版本**: v1.0
> **日期**: 2026-03-31

---

## 1. 产品概述

**ClawBar** 是一个 macOS menu bar 聊天客户端，让用户无需安装飞书、Discord、Teams 等重型应用，直接通过 menu bar 图标一键唤起轻量聊天窗口，与自部署的 OpenClaw（龙虾）实例进行对话。

### 1.1 目标用户

- 在 Azure VM 或其他服务器上自部署了 OpenClaw 的开发者
- 希望通过 OpenClaw 的 default channel 进行日常聊天
- 不想为聊天安装重型 IM 客户端
- 使用 macOS 的用户

### 1.2 核心价值主张

| 痛点 | ClawBar 的解决方案 |
|------|-------------------|
| 默认 channel 聊天界面丑 | 自定义美观的聊天 UI，支持 Markdown、代码高亮 |
| 聊天入口深藏在网页中 | macOS menu bar 一键唤起，随叫随到 |
| 不想装飞书/Discord/Teams | 极简单一入口，只做聊天 |
| 窗口不够灵活 | 支持 resize、拖拽、pin（置顶） |

## 2. 用户故事

### US-01: 快速唤起聊天
> 作为一个开发者，我想点击 macOS menu bar 上的 ClawBar 图标，立即弹出聊天窗口，这样我可以在任何工作场景下快速与 OpenClaw 对话。

**验收标准**:
- [ ] Menu bar 显示 ClawBar 图标（16x16 template icon）
- [ ] 点击图标，聊天窗口在图标下方弹出，动画时长 < 200ms
- [ ] 再次点击图标或按 ESC，窗口隐藏
- [ ] 应用不在 Dock 中显示

### US-02: 发送和接收消息
> 作为一个用户，我想在聊天窗口中输入文本并发送，然后看到 OpenClaw 的回复，支持 Markdown 格式和代码块。

**验收标准**:
- [ ] 输入框支持多行输入（Shift+Enter 换行）
- [ ] Enter 发送消息
- [ ] 用户消息显示在右侧气泡
- [ ] AI 回复显示在左侧气泡，支持 Markdown 渲染
- [ ] 代码块有语法高亮和一键复制按钮
- [ ] 发送后自动滚动到最新消息
- [ ] AI 正在回复时显示打字指示器

### US-03: 窗口灵活操作
> 作为一个用户，我想自由调整聊天窗口的大小和位置，并能将窗口置顶，这样我可以边工作边聊天。

**验收标准**:
- [ ] 窗口可通过拖拽标题栏移动位置
- [ ] 窗口可通过边缘拖拽调整大小（最小 320x400，最大 800x900）
- [ ] 标题栏有 Pin 按钮，点击后窗口置顶（always on top），图标状态变化
- [ ] Pin 状态下点击外部不会隐藏窗口
- [ ] 非 Pin 状态下点击外部可选隐藏窗口（可在设置中配置）

### US-04: 配置 OpenClaw 连接
> 作为一个用户，我想配置 OpenClaw 的连接方式，这样我可以连接到我的自部署实例。

**验收标准**:
- [ ] 首次启动显示配置引导
- [ ] 支持本地模式：自动检测本机 openclaw CLI 是否可用
- [ ] 可配置 openclaw 命令路径（默认从 PATH 中查找）
- [ ] 配置保存后持久化，下次启动自动连接
- [ ] 连接状态指示器（绿色=已连接/红色=断开/黄色=连接中）

### US-05: 会话管理
> 作为一个用户，我想能够创建新会话、切换历史会话，这样我可以管理多个对话主题。

**验收标准**:
- [ ] 标题栏显示当前会话名称/Agent 名称
- [ ] 点击标题栏会话名称弹出会话列表
- [ ] 可创建新会话（选择 Agent）
- [ ] 可切换到历史会话，加载历史消息
- [ ] 显示会话最后一条消息的预览

### US-06: 主题切换
> 作为一个用户，我想在亮色和暗色主题之间切换，以匹配我的系统外观。

**验收标准**:
- [ ] 支持亮色、暗色、跟随系统三种模式
- [ ] 主题切换即时生效，无需重启
- [ ] 默认跟随系统

## 4. 非功能需求

### 4.1 性能
- 应用启动到 menu bar 图标出现 < 2 秒
- 点击图标到窗口显示 < 200ms
- 消息发送到回复开始显示 < OpenClaw 处理时间 + 500ms
- 内存占用 < 150MB（空闲时）

### 4.2 安全
- 不存储任何 API 密钥（依赖 OpenClaw 自身认证）
- 本地 CLI 模式不暴露任何网络端口
- 配置文件使用 electron-store（加密存储敏感数据）
- 不收集任何用户数据

### 4.3 兼容性
- macOS 12+ (Monterey 及以上)
- Apple Silicon (arm64) + Intel (x64)
- OpenClaw 3.24+（首版）

### 4.4 可配置性
- 所有连接参数可配置（CLI 路径、端口等）
- 开源友好：其他用户 clone 后改个配置就能用

---

## 5. 技术约束

- 使用 Electron 38 构建
- 前端使用 React 19 + TypeScript + Tailwind CSS 4
- 通过 OpenClaw CLI（child_process.spawn）与 OpenClaw 通信
- 使用 `menubar` npm 包处理 menu bar 集成
- 打包为 macOS DMG（electron-builder）
