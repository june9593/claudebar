用 Python 写一个快速排序
OpenClaw
当然，这是一个简洁的实现：
def qsort(arr):
  if len(arr) <= 1: return arr
  p = arr[0]
  return qsort([x for x in arr[1:] if x<p]) + [p] + qsort([x for x in arr[1:] if x>=p])
时间复杂度平均 O(n log n)，最坏 O(n²)。需要优化版本吗？
