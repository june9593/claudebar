# clawbar – OpenClaw Menu Bar Chat UI (static design)

This folder contains **static HTML mockups** (no backend) for a modern-looking OpenClaw “menu bar entry → quick chat” experience.

## Open the designs

Double-click any `.html` file (or serve locally) to preview:

- `index.html` – launcher/gallery to switch between concepts
- `concept-a-glass.html` – glassmorphism, macOS-like popover
- `concept-b-sidebar.html` – 2-pane “control center” layout
- `concept-c-minimal.html` – ultra-compact quick chat

## Notes

- These are purely UI/UX mockups: buttons don’t actually call OpenClaw.
- Typography uses system fonts; icons are inline SVG.
- Dark-mode first, with light mode support.
