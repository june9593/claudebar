帮我写一个快速的 python 脚本，统计当前目录下所有 .js 文件的总行数。

好的，这里是一个使用 pathlib 快速统计的脚本：

count_lines.py 
import os
from pathlib import Path

total = 0
for p in Path('.').rglob('*.js'):
    with open(p, 'r', encoding='utf-8') as f:
        total += len(f.readlines())
print(f"Total lines: {total}")
